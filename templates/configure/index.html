Простой режим сборки

Данный режим предназначен для автоматической оптимальной конфигурации персонального компьютера, исходя из вашего бюджета и целей данного устройства.

Расширенный режим сборки

Данный режим предназначен для полуавтоматической оптимальной конфигурации персонального компьютера. Здесь Вы можете задать как бюджет в целом, так и предпочитаемые ценовые диапазоны для каждой из комплектующих, а так же и другие их параметры.

Ручной режим сборки

Данный режим предназначен для ручного подбора комплектующих. Пользователь самостоятельно осуществляет выбор комплектующих из списка доступных и удовлетворяющих условиям совместимости.

Конфигуратор ПК

Это онлайн-конструктор, с помощью которого можно самостоятельно или автоматически подобрать комплектующие для будущего компьютера или обновить уже существующую систему.
